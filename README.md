# Dear Future - La Capsule Temporelle Numérique

## À propos
Ce projet est développé dans le cadre du module Programmation PHP et Framework à l'**EMSI**. Il s'agit d'une application web de capsule temporelle permettant aux utilisateurs d'écrire des lettres qui ne se déverrouillent qu'à une date future précise, garantissant la confidentialité et l'intégrité du message jusqu'au moment choisi.

## Fonctionnalités
* **Capsule Temporelle :** Rédaction de lettres sécurisées inviolables avant la date de délivrance.
* **Sécurité Temporelle :** Verrouillage strict géré côté serveur (Controller) empêchant l'accès technique avant la date, couplé à un masquage visuel côté client.
* **Expérience Visuelle :** Système de "Smart Zoom" développé en CSS pour une sélection immersive des cadeaux associés.
* **Espace Membre :** Gestion complète des utilisateurs (Inscription, Authentification) et archivage des lettres.

## Technologies
* **Backend :** PHP 8.2, Symfony 7
* **Frontend :** Twig, HTML5, CSS3
* **Base de données :** MySQL, Doctrine ORM
* **Sécurité :** Symfony Security Bundle
